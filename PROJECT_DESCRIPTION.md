Goal: To build a NeoVim like Text Editor that works agnostic to the os and works like neovim without requiring users to install
fonts or packages or needing a specific os.

Necessary:
Must be able to use vim macros
Move like you're in vim.
Plugin in LSPs and Syntax Highlighting
Must be able to be expanded by users for customiztion
Must allow for maximum user fluency and handle all inconvient things and allow users to fly


Coding language:
Built in GO.
Uses allows multiple languages for scripting including lua, python, and carrion and customization.
